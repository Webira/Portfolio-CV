Mes réalisations

« Professors management »

Pour la période de stage dans la Société Kebtechs, j'ai realisé le projet d’utilisation interne pour l’administrateur : Tableau de la gestion des professeurs « Professors management » - une application de type Desktop.

J’ai développé une API REST côté serveur avec le Framework Spring Boot pour Back-end avec MySQL pour la base de données, ainsi qu’une interface côté client pour afficher un résultat «visuel» fait avec REACT JS, en utilisant les langages : JAVA, HTML, JS, CSS

Le but d’application Tableau « Professors-management » :

• Effectuer l'enregistrement des données statistique des Professors , pour l'analyse et la gestion simplifiée, avec installation sur l’ordinateur de l’administrateur.

« Manager open Simulateur »

Realisé pour l’Association MEZA le développement du projet « Manager open Simulateur » : authentification et autorisation des utilisateurs, gestion d’utilisateurs d’Open Simulateur.

Les buts de l'application Web sont :

• D’effectuer l'enregistrement et l’authentification des Utilisateurs, selon les trois rôles prévus, pour les connecter en suite avec le Framework open simulator, qui rendre accessible au plus grand nombre l'environnement de réalité virtuelle.

• Développer un Blog, pour garder des contacts interactifs avec les utilisateurs et pour améliorer le positionnement l’application sur les pages de résultats de recherche (SEO).

• De simplifier le processus de la transmission de données statistiques à l’administration de l’association qui les analyse et utilise pour la gestion utilisateurs.

En résumé :

Pour concevoir et développer une application en PHP avec Architecture MVC et des modèles de conception : Le Factory Pattern et La chaîne de responsabilité, j’ai travaillé en mode Agile (SCRUM) depuis l’analyse de l’expression des besoins clients jusqu’au développement d’une solution logicielle de gestion des utilisateurs basée sur une architecture N-Tiers, implémentée en langage PHP. Pour ce faire, je vais concevoir une base de données relationnelle pour stocker les informations (utilisateurs, contenus spécifiques), et une API REST pour la gestion.

Pour développer la partie Front-end et la partie Back-end d’application, en intégrant les recommandations de sécurité, j’ai utilisé les langages : HTML, JS, PHP, CSS ainsi que Framework Laravel 8 et Bootstrap 5, Base de données MySQL

Lien du projet : Back-end et Front-end

« Plateforme d'inscription »

Le développement du projet "Plateforme d'inscription en ligne" pour une école privée - l'Association « En mode bilingue ». J’ai travaillé en mode Agile (SCRUM) depuis l’analyse de l’expression des besoins clients jusqu’au développement d’un solution logicielle : d’inscription d’élevés à l’école et d’une gestion.

Développement la partie front-end et la partie back-end d’application WEB, en intégrant les recommandations de sécurité, a effectué avec: HTML, JS, PHP, CSS ainsi que Frameworks Symfony et Bootstrap, Base de données MySQL et la libraire JQuery.

Les buts de l'application « Plateforme d'inscription » sont :

• D’effectuer l'inscription des élèves aux ateliers créatifs par internet, en ligne. Pour simplifier le processus administratif pour les parents des élèves ;

• De simplifier le processus de la transmission de données statistiques à l’administration de l’association qui les analyse et utilise pour la gestion. Les tableaux administratifs responsifs.
